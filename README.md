## 🎯 A Professional, Responsive Digital Resume Built with React.js

A modern, interactive digital resume showcasing my skills, experience, education, and projects. Built with React.js and deployed on Surge.

### 🌐 **Live Demo**
👉 **[View My Live Resume](https://hatesh-resume.surge.sh)** 👈

---

## ✨ **Features**

| Feature | Description |
|---------|-------------|
| 📱 **Fully Responsive** | Works perfectly on Desktop, Tablet, and Mobile devices |
| 🖼️ **Profile Image** | Professional profile picture with hover effects |
| 📊 **Animated Skill Bars** | Visual representation of technical skills with smooth animations |
| 📄 **PDF Download** | One-click download button to save resume as PDF |
| 🔗 **Social Links** | Direct links to GitHub and LinkedIn profiles |
| 🎨 **Modern UI** | Gradient backgrounds, shadows, and smooth transitions |
| 🖨️ **Print Friendly** | Optimized for printing or saving as PDF |
| 📅 **Dynamic Footer** | Automatically updates copyright year |
| ⚡ **Fast Performance** | Built with Vite for optimal loading speed |

## 🚀 **Getting Started**

### **Prerequisites**
- Node.js (v16 or higher)
- npm or yarn

### **Installation**

1. **Clone the repository**
```bash
git clone https://github.com/hatesh1/react-resume-portfolio.git
cd react-resume-portfolio
```
### **Install dependencies** 
- npm install

### **Start development server**
- npm run dev
